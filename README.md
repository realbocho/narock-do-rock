# 3D 행운 돌탑 🗿

3D로 구현된 인터랙티브 행운 돌탑 웹사이트입니다. 소원을 담은 돌을 쌓아 나만의 행운 돌탑을 만들어보세요!

## ✨ 주요 기능

- **3D 돌탑 시각화**: Three.js를 활용한 실감나는 3D 돌탑
- **인터랙티브 돌 쌓기**: 다양한 모양과 크기의 돌 선택 가능
- **소원 작성**: 각 돌에 개인적인 소원이나 메시지 작성
- **데이터 저장**: Supabase를 통한 돌탑 데이터 영구 저장
- **통계 기능**: 전체 돌 개수, 오늘 쌓인 돌 개수 표시
- **반응형 디자인**: 모바일과 데스크톱 모두 지원
- **햅틱 피드백**: 모바일에서 진동 피드백 제공

## 🚀 배포 및 설정

### 1. Supabase 설정
1. [Supabase](https://supabase.com)에서 새 프로젝트 생성
2. `SUPABASE_SETUP.md` 파일의 SQL 코드를 실행하여 테이블 생성
3. `public/index.html`에서 Supabase URL과 anon key 설정

### 2. Vercel 배포
1. GitHub에 프로젝트 업로드
2. [Vercel](https://vercel.com)에서 GitHub 연동
3. 자동 배포 완료

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **3D Graphics**: Three.js
- **Database**: Supabase (PostgreSQL)
- **Deployment**: Vercel
- **Animation**: CSS Transitions & Three.js Animations

## 📁 프로젝트 구조

```
stone/
├── public/
│   └── index.html          # 메인 HTML 파일
├── package.json            # 프로젝트 설정
├── vercel.json            # Vercel 배포 설정
├── SUPABASE_SETUP.md      # Supabase 설정 가이드
└── README.md              # 이 파일
```

## 🎮 사용법

1. **웹사이트 접속**: 배포된 URL로 접속
2. **돌 추가**: 우하단의 '+' 버튼 클릭
3. **소원 작성**: 소원이나 메시지 입력
4. **돌 설정**: 모양(둥근돌/타원돌/네모돌)과 크기(작게/보통/크게) 선택
5. **돌 쌓기**: 확인 버튼을 눌러 돌탑에 추가
6. **돌탑 감상**: 3D로 구현된 나만의 돌탑 확인

## 🎨 시각적 특징

- **동적 배경**: 돌 개수에 따라 배경색 변화
  - 0-4개: 하늘색 그라데이션
  - 5-9개: 노을 그라데이션  
  - 10-19개: 보라색 그라데이션
  - 20개 이상: 어두운 그라데이션

- **물리 효과**: 돌이 떨어질 때 자연스러운 중력과 바운스 효과
- **그림자**: 실감나는 그림자 효과로 입체감 강화
- **텍스트 렌더링**: 돌 표면에 소원 텍스트 표시
